# Full-Stack Online Course Platform - Synent Technologies Task 8

A complete MERN-stack learning management application featuring robust user account authentication, course catalogs, admin content structures, and sandbox payment flows.

## ⚡ Backend Capabilities Completed
- **Authentication**: JWT validation tokens and salted password generation using bcrypt.
- **Content Hierarchy**: Node database maps for Courses -> Modules -> Lessons structures.
- **Payment Gateway**: Test-mode SDK payment integrations using Razorpay order tracking APIs.

## 🛠️ Stack Components
- Node.js & Express.js
- MongoDB & Mongoose
- Razorpay Web Tokens SDK
